이번에는 실제 면접장의 분위기가 조성된 실전 면접 기능을 이용해보시는게 어떨까요?

Record

  • 1.

    내역 페이지 접속

    내역 페이지에 접속하시면 다음과 같이 현재까지 완료한 모의면접 내역을 확인하실 수 있습니다!

  • 2.

    편리한 내역 정렬 기능

    내역이 너~무 길어서 이전 기록을 확인하기 힘드시다구요? 우측 상단에 위치한 정렬 기능을 이용해보세요! 연습면접 버튼을 클릭하면 연습면접결과만! 실전면접 버튼을 클릭하면 실전면접결과만 확인하실 수 있습니다. 다시 모든 결과를 확인하시려면 전체 버튼을 눌러주세요.

  • 3.

    이전 결과 확인하기

    다시 확인하고 싶은 내역을 클릭하면 상세 분석 결과를 다시 확인해보실 수 있습니다.
